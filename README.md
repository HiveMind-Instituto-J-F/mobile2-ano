# Hive Mind – Mobile

 O app atua como ponto central de coleta de dados de micro paradas, apoiando decisões operacionais e aumentando a eficiência.

---

## Visão Geral

O app mobile permite:

- Registro rápido de microparadas
- Consulta e histórico de ocorrências
- Integração direta com API REST
- Visualização de informações de produção
- Suporte a ambientes industriais e operacionais
- Notificação no momento em que o status da microparada for atualizado

---

## Funcionalidades

-  Login e autenticação
-  Registro de microparadas
- Histórico e consulta de registros
- Integração com backend via API
-  Sincronização com banco de dados
- Dashboard 

## Tecnologias Utilizadas

| Categoria | Tecnologia |
|----------|------------|
| Linguagem | Java |
| Framework | Android SDK |
| Arquitetura | MVVM |
| Rede / API | Retrofit |
| Banco de Dados (via API) | PostgreSQL / MongoDB |
| UI/UX | Material Design |
| Versionamento | Git / GitHub |
